### 精選研究與專案

#### 1. 每日論文翻譯推薦系統 (Marker-Gemini-Translator)
[**[檢視專案]**](https://github.com/Bookweight/Marker-Gemini-Translator.git)
> **主題：** 自動化 AI 研究助手
* **功能**：一套自動化系統，用於抓取、推薦並翻譯每日研究論文。
* **核心概念**：將原有的 Shell/Python 混合流程重構為 **純 Python 架構**。整合 **Google Gemini SDK** 以獲得高語境感知的翻譯品質，並使用 **Marker** 進行高保真 PDF 轉 Markdown 處理。
* **技術堆疊**：`Python` `Google Gemini` `Marker` `Automation`

#### 2. 基於聯邦學習的惡意軟體分類
[**[檢視專案]**](https://github.com/Bookweight/Malware-classification-using-federated-learning.git)
> **主題：** 隱私保護 AI 與電腦視覺
* **核心架構**：使用多模態輸入對病毒原始碼（灰階圖）及 API 呼叫序列（彩色圖）進行分類。
* **核心概念**：實作結合 **差異隱私 (Differential Privacy)** 的 **聯邦學習 (Federated Learning)** 架構，以防禦後門攻擊（數據投毒）。
* **技術堆疊**：`Python` `PyTorch` `CUDA` `Federated Learning`

#### 3. 玉山人工智慧公開挑戰賽：警示帳戶偵測
[**[檢視專案]**](https://github.com/Bookweight/AICup-AML-2025-Team8482.git)
> **主題：** 金融詐欺偵測 (前 4% 成績)
* **成就**：在 790 支隊伍中排名 **第 30 名**。
* **技術細節**：處理包含交易明細的高度 **不平衡資料集**。透過進階特徵工程優化模型以最大化 F1-score。
* **技術堆疊**：`Data Mining` `Imbalanced Learning` `Scikit-learn`

#### 4. 社群媒體熱門文章預測器
[**[檢視專案]**](https://github.com/Bookweight/Social-Media-Trending-Post-Predictor.git)
> **主題：** 即時數據分析
* **功能**：透過自製爬蟲抓取即時 PTT 數據，預測未來 2 小時內的 **前 10 大熱門文章**。
* **表現**：達成 **NDCG@10 0.97 的準確率**。
* **技術堆疊**：`Python` `Web Crawler` `Time-series Prediction`
